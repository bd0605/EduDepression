# 學生憂鬱症分析 - 簡易說明

版本：v1.0.0  
日期：2025-05-11

## 專案目的

分析不同學歷學生的憂鬱症風險

## 快速執行

```bash
# 1. 安裝套件
pip install -r requirements.txt

# 2. 執行分析
python run_analysis.py
```

## 主要功能

- 資料載入與清洗
- K-means 聚類（3 群組）
- 邏輯回歸預測
- 基礎視覺化

## 輸出結果

- 統計圖表：`visuals/`
- 分析報告：`report/`
