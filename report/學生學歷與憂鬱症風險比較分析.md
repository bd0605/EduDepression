# 學生學歷與憂鬱症風險比較分析

版本：v1.0.0  
更新日期：2025-05-11

## 摘要
本報告使用 XAMPP + MySQL + Grafana 架構，分析不同學歷學生的憂鬱症風險。採用 K-means 聚類和邏輯回歸方法，探討學業壓力、工作壓力等因素對憂鬱症的影響。
